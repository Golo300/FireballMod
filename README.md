# FireballMod

A Minecraft 1.8.9 Forge mod that adds a scope with zoom functionality.

## Features

- **Scope Overlay** - Clean sniper scope interface
- **Variable Zoom** - Mouse wheel to adjust zoom level
- **Distance Display** - Shows distance to targeted block
- **Camera Shake** - Recoil effect when firing
- **Clean HUD** - Hides crosshair and hotbar while scoped

## Installation

1. Download the latest release from [Releases](https://github.com/yourusername/FireballMod/releases)
2. Install Minecraft Forge 1.8.9
3. Place the `.jar` file in your `.minecraft/mods` folder
4. Launch Minecraft with Forge 1.8.9

## Building
```bash
git clone https://github.com/yourusername/FireballMod.git
cd FireballMod
./gradlew build
```
